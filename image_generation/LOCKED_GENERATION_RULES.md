# 固定計画による社会科バッチ生成規則

固定計画が存在する場合、`LOCKED_PAGE_SEQUENCE.json`と指定された`public_batches/BATCH_XX.md`が唯一の正本である。

## バッチ指定時

1. 指定`BATCH_XX.md`の10件だけを読む。
2. 登録されたMarkdown、JSON、直接参照素材だけを確認する。
3. 別バッチ・未登録素材を探索しない。
4. 地図・年表・写真・表・グラフは直接貼らず、事実関係を独自に再構成する。
5. Image 2.0を10回に分け、一枚ずつ生成する。

## 社会科固有の注意

- 年号、人名、地名、制度、出来事の前後関係を正確にする。
- 地図上の方位、地域、国境、行政区分を誤らない。
- 統計値は基準年を明記し、元資料と異なる現在値へ更新しない。
- 歴史資料は対象時代を明示し、現代の地名や制度と混同しない。
- 政治制度や選挙制度を推測で補わない。

## 絶対条件

- 10枚を1枚にまとめない
- 1回につき1枚だけ生成する
- 横9：縦16
- 1080×1920相当
- PNGラスター画像
- ベクター禁止
- コラージュ、一覧、後分割禁止
- 中学受験重要語句は`#D71920`、太字、同色実線下線
- 穴埋め、選択肢、クイズ、タップUIは禁止

## 文字の読みやすさ（2026-07-16追加）

- 1080×1920換算で、本文は34px相当未満にしない。
- 地図・グラフのラベルは30px相当未満にしない。
- 中学受験の頻出語句・重要数値は48〜64px相当、太字、`#D71920`または濃い青で表示する。
- タイトルは画面高の8〜10％を使い、ひと目で読める大きさにする。
- 1画像は「大図解1つ＋説明3〜5ブロック」まで。細かい表、長文注釈、8項目以上の一覧は禁止する。
- 文字が収まらない場合は縮小しない。理解単位を再分割して別画像へ送る。
- キャラクターは画面面積の18％以内を目安にし、本文や地図の面積を圧迫しない。
