# GitHubコネクターなしでの公開Web実行

このリポジトリは公開される前提である。GitHubコネクターが利用できないことだけを理由に停止してはいけない。

固定計画生成後は、次を通常のWebブラウズで開く。

- `https://github.com/hiroakiyasa/shakai/tree/main/image_generation/public_batches`
- `https://github.com/hiroakiyasa/shakai/blob/main/image_generation/public_batches/BATCH_XX.md`

各`BATCH_XX.md`には、固定10件のMarkdown、JSON、直接参照素材、キャラクターURL、生成規則が含まれる。

## 禁止する応答

- GitHubコネクターがないため実行できない
- ファイルをアップロードしてください
- 固定10件を貼り付けてください

公開Webの`BATCH_XX.md`が存在する場合は、その内容で処理を続行する。

## 公開Web実行の範囲

- 指定されたBATCH_XX.mdだけを読む
- そのファイルに登録された10件だけを確認する
- 登録済みMarkdown・JSON・直接参照素材以外を探索しない
- 全教材の再監査、再配分、画像数変更、バッチ数変更を行わない
